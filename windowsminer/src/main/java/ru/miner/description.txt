Для реализации приложение Windows Miner, будем использовать паттерн "Стратегия". Принцип проектирования построен на использовании
композиции в приложении. С помощью композиции можно безболезненно добавлять любые модули. Каждый модуль между собой должен иметь
слабую связь. Для этого используется связка интерфейс (перечислены методы) - класс (реализация методов)
В данной программе реализовано в папке Fields:
Field - поле и ее методы
Cell- ячейка и ее методы
CreateField- создания поля
В папке Logic описывается логика игры, состоит из интерфейса Logic и реализации класса LogicImpl
В папке UserActivity реализуется действия пользователя.
